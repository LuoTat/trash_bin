## 疑问

**1. 如何安装该软件仓库中的软件？**

确保你已经有 Scoop 环境，执行以下命令订阅本软件仓库：

```powershell
scoop bucket add trash_bin https://github.com/LuoTat/trash_bin
```

执行以下命令安装本仓库中的软件：

```powershell
scoop install trash_bin/<软件名>
```

**2. 我想要某个软件，这个仓库里没有！**

开 [issue]，描述你的需求。:)

**3. 仓库中的某个软件版本落后了，求更新！**

欢迎 Fork 本仓库，修改落后的软件清单，并提交你的拉取请求。:D
